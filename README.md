# Personal Deadline Tracker (Specifically For studs)

A simple and effective frontend web app to help students manage courses, track assignments, and never miss deadlines again.


---

## Overview

The **Personal Deadline Tracker** is designed to solve a common problem:  
> Students often miss deadlines because assignments are scattered across different platforms.

This app brings everything into one place, allowing users to:
- Organize courses
- Track assignments
- Monitor deadlines visually

---

## Why I Built This

As a student, I noticed that my assignments and deadlines were always scattered across different places class notes, messaging apps, and random reminders.  

It became easy to forget important tasks, especially during busy weeks.

Instead of relying on multiple tools, I wanted a **simple and focused solution** where I could:
- See all my deadlines in one place
- Know what’s urgent at a glance
- Stay more organized without extra complexity

So I built this project not just to practice frontend development, but to solve a real problem I personally face.

---

## My Goal

This project helped me move beyond just writing code and start thinking like a developer who builds **useful and user-centered applications**.

I focused on:
- Creating a clean and intuitive UI
- Making features practical and not overcomplicated
- Improving user experience with small but meaningful details

## Features

- Add and manage courses
- Add assignments with deadlines
- Deadline countdown (e.g. "2 days left", "Due Today")
- Priority levels (Low / Medium / High)
- Filters (All / Today / This Week)
- Delete assignments
- Edit assignments
- Dark mode toggle
- Data persistence using Local Storage
